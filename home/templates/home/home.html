Welcome

This is a To-do app, its features are as follows:

  • The user can add Todo lists, corresponding Tasks and Comments with File Attachments.
  • The user can if registered and logged in Delete, View or Update any of their user data.
  • All communication after Login happens through Ajax calls.
  • Tasks with a a background colour of lightsalmon are high priority and lightcyan indicates normal priority tasks.
  • Tasks with a strike-through indicated that they have been completed.
